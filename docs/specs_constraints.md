# Card Shuffling & Dealing Machine Specification and Constraints (7/23 updated)

**Purpose:** Build an automated, reliable playing-card machine that can shuffle and pass cards to players.

---

### Constraints / Specs:

i) Mechanical assembly, routing infrastructure, and electronics must be housed cleanly inside an enclosure box.  
ii) Machine operates on a single battery pack (voltage of choosing).  
iii) Must be portable and weigh no more than 1.4 kg (3 lbs).  
iv) Must have a user interface for menu navigation.  
v) Needs functions to perform the following:  
   a) Shuffling within 2-4 seconds 
   b) Dispense a specified # of cards to all players (0.5-1.0 sec / card dispense intervals)
   c) Dispense a card to a player based on user input
vi) Any electronic hardware can be used, minimize expenses on project as much as possible
vii) Incorporate thermal cooling/venting in enclosure design  
viii) Microcontroller USB port should be accessible from outside the box.  
ix) Power switch must be accessible and labeled.  
x) Prevent system from running on too low battery voltage, Automatic turn off
xi) Must be within an 10" x 10" x 10" box
